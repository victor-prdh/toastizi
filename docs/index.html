Toastizi

Minimalist and easy to use toast plugin written in pure JS.

You can try the plugin with the button below.

Open a basic toast
Open a toast with close button
Open a styled toast

How to integrate Toastizi in your projects ?


First you need to download it. You have two possibilities:

  • Download the minified files here.
  • You need to edit files ? Go to the GitHub repository and download the files in the src folder.


Now you can create a new toast like that


        var toast = new toastizi.toast({
            content: 'The content of the toast',
            closeButton: true,
            autoClose: true,
            autoCloseTime: 8000,
            backgroundColor: "#fff",
            color: "#000",
          });

You have just to ask him to appear like that:


        toast.open();

You want close him with javascript ? It's like that:


        toast.close();
